一時変数を使わずに与えられた数値を入れ替えるメソッドを作成してください。

ただし、メソッド内で変数を定義することは禁止です。

メソッドは以下の表を基に作成すること。

関数名:swap

仮引数1:(int) a

仮引数2:(int) b

返り値:なし

実行例

public class Kadai004 {

    public static void main(String[] args) {
        int a = 3;
        int b = 5;
        swap(a, b);
    }

    /* ここにswapメソッドを定義 */
}

出力結果

a: 5
b: 3
